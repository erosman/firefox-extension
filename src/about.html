Release Notes for Recent Releases

Versions 7.0

  • Option to set active on|off on proxy Edit page
  • username/password are kept when adding new proxies for rapid creation of multiple proxies with same username/password
  • Log shows local date & time in local (browser) language
  • Log is not cleared when switching it off until ti is manually cleared
  • Log entries are numbered
  • Option to transfer preferences from local to Sync and vice versa
  • Options page shows for proxies with username/password
  • Date is added to the export filename
  • Export auto-renames file if the same name exists
  • Import data max size set to 5mb
  • Mouseove on ellipsised text on patterns shows the full text
  • Greatly streamlined
  • Style/Code Improvement

Versions 6.4, 6.5, 6.6, 6.6.1, 6.6.2 (some versions unreleased)

  • Fixed pattern matching bug whereby patterns with leading .* or . did not always accurately match subdomains
  • Import/Export patterns to and from files (previously you could only import/export all settings, including patterns)
  • Replaced paging on the patterns page with infinite scroll
  • Replaced the inline editing on the patterns page with a modal dialog
  • Several UI improvements; for example, rounded containers are now less round, message after successful export, etc.
  • Removed all minified code for Mozilla; change filenames of 3rd-party libs to include version number
  • Security fixes with jQuery
  • Minor UI fixes to the toolbar popup window
  • Initial i18n support with 1 page localized
  • Removed several jQuery plugins towards the day when jQuery will be completely removed. Upgraded jQuery from 2.2.4 to 3.4.1 per Mozilla regulations.
  • Removed cruft
  • Minor updates to About and First Install pages
  • Added link on About Page to the now public github repository for source code

Version 6.3

  • Turn Firefox Sync on/off - much requested feature.
  • Some further input validation

Version 6.2

  • Ability to turn logging on/off -- much requested feature that was in legacy FoxyProxy
  • New icon on main page for "What's my IP address?", previously only in the About page.
